Step 0. Build Murphi

tar -xvf eecs570_p2.tar.gz
cd Murphi3.1/src
make mu

Step 1. Compile Murphi source

cd Murphi3.1/eecs570_sample
./mu msi.m
./mu msi_opt.m

This will generate a C source file named twostate.C

Step 2. Compile C code generated by Murphi

make msi
make msi.opt

Step 3. Run

./msi
./msi_opt
